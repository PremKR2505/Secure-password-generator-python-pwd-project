# Secure Password Generator:Python pwd project

A password generator with user defined constraints such as length and keywords

## Features

- User can define constraints and also put a keyword
- Set desired password length
- Include/exclude:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special symbols (!@#$%^&* etc.)
- Option to **exclude ambiguous characters** (e.g., `l`, `1`, `I`, `O`, `0`)
- Copy with one click
